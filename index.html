CURVETOPIA: A Journey into the World of Curves

Welcome to Curvetopia, where we bring order and beauty to the world of 2D curves!

Objective

Our mission is to identify, regularize, and beautify curves in 2D Euclidean space. We start with closed curves and progressively work with more complex shapes. The project also covers symmetry and curve completion techniques.

Problem Description

The goal is to convert a PNG (raster) image of line art into a set of curves defined as cubic Bézier curves. Initially, we simplify the task by using polylines, which are sequences of points. The challenge involves regularizing, symmetrizing, and completing these curves.

Approach

1. Regularize Curves

Identify regular shapes such as:

  • Straight lines
  • Circles and Ellipses
  • Rectangles and Rounded Rectangles
  • Regular Polygons
  • Star Shapes

2. Exploring Symmetry in Curves

Identify symmetry in closed shapes, focusing on reflection and rotational symmetries.

3. Completing Incomplete Curves

Develop algorithms to complete gaps in curves caused by occlusions, with varying levels of complexity.

Visualization

Results are visualized using SVG format, and output curves are converted to cubic Bézier curves. Sample visualizations and code snippets are provided for reference.

Sample Code

Reading CSV Files

import numpy as np
def read_csv(csv_path):
    np_path_XYs = np.genfromtxt(csv_path, delimiter=',')
    path_XYs = []
    for i in np.unique(np_path_XYs[:, 0]):
        npXYs = np_path_XYs[np_path_XYs[:, 0] == i][:, 1:]
        XYs = []
        for j in np.unique(npXYs[:, 0]):
            XY = npXYs[npXYs[:, 0] == j][:, 1:]
            XYs.append(XY)
        path_XYs.append(XYs)
    return path_XYs
